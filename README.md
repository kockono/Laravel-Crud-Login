## Metodo De Instalación Normal
___
1. Entrar a src 
```
cd src
```
2. Instalar las librerias
```
npm install
```
3. Correr y cargar las librerias
```
npm run dev
```
4. Correr el servidor
```
php artisan serve
```
<br>

## Instalación Con  Dockers
___
*Crear Contenedor*
```
docker-compose up -d
```
*Eliminar Contenedor*
```
docker-compose down
```
